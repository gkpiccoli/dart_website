## Como rodar o projeto:

## Running and building

- Clone o repositório.
- No terminal, execute: 

  ```console
  $ dart pub get
  ```

## O que eu aprendi:

- Dividir o conteúdo de um site em containers para melhorar a visualização da página;
- Usar fontes externas;
- Criar um servidor Dart que roda HTML/CSS;
